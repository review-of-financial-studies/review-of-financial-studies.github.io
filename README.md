# Review of Financial Studies - Data Editors Website

The public website of the Data Editors of the *Review of Financial Studies*.

**Live site:** [https://review-of-financial-studies.github.io/](https://review-of-financial-studies.github.io/)

## About

This repository contains the source files for the RFS Data Editors website, which provides guidance and resources for authors submitting replication packages to the *Review of Financial Studies*.

## Built With

- [Quarto](https://quarto.org/) - An open-source scientific and technical publishing system

## Local Development

1. Install [Quarto](https://quarto.org/docs/get-started/)
2. Clone this repository
3. Run `quarto preview` to start a local development server
4. Run `quarto render` to build the site
